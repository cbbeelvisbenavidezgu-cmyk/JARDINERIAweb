# 🌿 Logee's Plants for Home & Garden

Página web moderna, responsive y funcional para una tienda de plantas desarrollada con **HTML5, Tailwind CSS y Vanilla JavaScript**, en un solo archivo (`index.html`) completamente autónomo.

Incluye:
- Landing page moderna
- Carrusel hero automático
- Catálogo de productos dinámico
- Carrito de compras funcional
- Sistema de checkout con formulario
- Modo oscuro persistente
- Diseño responsive mobile-first
